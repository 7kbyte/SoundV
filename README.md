# 🎵 桌面音频波形可视化器

实时捕获 Windows 系统音频，在桌面左下角显示炫酷的波形/频谱。

## ✨ 特性

- **系统音频采集** — 通过 WASAPI Loopback 捕获扬声器输出（非麦克风）
- **全透明覆盖** — 背景完全透明，鼠标事件穿透，不干扰正常操作
- **常驻置顶** — 窗口始终在桌面左下角，不被其他窗口遮挡
- **双显示模式** — 📈 波形（Catmull-Rom 平滑曲线）/ 📊 频谱（48 柱 FFT）
- **彩虹色相** — 波形线条从左到右渐变，频谱柱低到高彩虹着色
- **自动换色** — 色相缓慢旋转（可开关），让视觉不单调
- **光晕背景** — 波形峰值处径向光晕，层次感丰富
- **系统托盘** — 右下角图标，右键菜单控制显示/模式/颜色/退出

## 🚀 快速开始

### 运行可执行文件

下载 `dist/AudioVisualizer.exe`，双击运行（无需安装 Python）。

### 从源码运行

```bash
# 1. 创建虚拟环境
python -m venv .venv
.venv\Scripts\activate

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动
python main.py
```

### 打包

```bash
build.bat
```

## 🖱️ 使用方式

| 操作 | 效果 |
|------|------|
| 右键托盘图标 | 打开菜单 |
| 双击托盘图标 | 显示/隐藏波形 |
| 点击关闭按钮 | 隐藏到托盘 |

### 托盘菜单

- 🔊/🔇 **显示/隐藏** — 切换波形窗口
- 📈/📊 **波形/频谱** — 切换显示模式
- 🔄 **自动换色** — 开关色相自动旋转
- 🎨 **波形颜色** — 弹出彩虹色条选色
- ❌ **关闭插件** — 完全退出

## 📁 项目结构

```
mm/
├── main.py              # 程序入口
├── config.py            # 所有可调参数
├── audio_capture.py     # WASAPI Loopback 音频采集
├── color_picker.py      # 彩虹色选择弹窗
├── tray_manager.py      # 系统托盘 & 右键菜单
├── waveform_widget.py   # 透明波形窗口（核心）
├── requirements.txt     # Python 依赖
├── run.bat              # 开发启动脚本
├── build.bat            # PyInstaller 打包脚本
└── dist/
    └── AudioVisualizer.exe  # 可分发的可执行文件
```

## 🛠️ 技术栈

- **Python 3.10+**
- **PyQt5** — GUI 框架（透明窗口 + 系统托盘）
- **soundcard** — WASAPI Loopback 音频捕获
- **NumPy** — FFT 频谱分析 + 波形处理
- **PyInstaller** — 打包为独立 .exe
